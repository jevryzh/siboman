# 逐梦 ERP 采集端插件

这是 Ozon-1688 ERP 的浏览器插件版采集端。

## 当前版本

`0.1.0` 是连接预览版：

- 支持在插件里登录 ERP 账号
- 支持保持在线心跳
- ERP 网页端可以看到当前浏览器插件在线
- 暂不自动领取正式采集任务，避免预览版误领任务

## 安装测试

1. 打开 Chrome 或 Edge。
2. 地址栏输入 `chrome://extensions/`。
3. 打开开发者模式。
4. 点击“加载已解压的扩展程序”。
5. 选择本目录：`browser-extension/ozon-1688-collector-extension`。
6. 点击插件图标，输入 ERP 地址、账号和密码。
