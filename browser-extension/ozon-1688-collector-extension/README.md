# 逐梦 ERP 采集端插件

这是 Ozon-1688 ERP 的浏览器插件版采集端。

## 当前版本

`0.4.0` 是真实采集版：

- 支持在插件里登录 ERP 账号
- 支持保持在线心跳
- ERP 网页端可以看到当前浏览器插件在线
- 支持手动开启“领取 ERP 采集任务”
- 支持领取单品找货和批量采集任务
- 单品找货支持 Ozon 采集、1688 真实以图搜货、候选详情补采和服务器 AI 审核

使用 1688 以图搜货前，请先在当前 Chrome/Edge 浏览器里登录 1688。插件会复用浏览器 Cookie 获取 1688 搜图 token。

## 安装测试

1. 打开 Chrome 或 Edge。
2. 地址栏输入 `chrome://extensions/`。
3. 打开开发者模式。
4. 如果下载的是 zip，请先解压，不要直接把 zip 拖进 Chrome。
5. 点击“加载已解压的扩展程序”。
6. 选择解压后包含 `manifest.json` 的文件夹。
7. 点击插件图标，输入 ERP 地址、账号和密码。
8. 需要执行采集时，打开“领取 ERP 采集任务”。
