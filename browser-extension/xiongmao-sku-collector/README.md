# Ozon SKU 数据采集器

这是一个本地浏览器插件，用来采集熊猫上架和 MY 页面里的 SKU/商品编号及表格列数据。

支持页面：

- `system.xiongmaoshangjia.cn`
- `my.jizhangerp.com`

## 安装

1. 打开 Chrome 或 Edge。
2. 地址栏输入 `chrome://extensions/` 并回车。
3. 打开右上角的「开发者模式」。
4. 点击「加载已解压的扩展程序」。
5. 选择这个文件夹：

   `/Users/eason/Documents/OZON/browser-extension/xiongmao-sku-collector`

如果之前已经加载过旧版插件，在扩展程序页面点击这个插件卡片上的刷新按钮，让浏览器读取新的 MY 网站权限。

## 使用

1. 打开熊猫上架热度排行，或 MY 的榜单选品页面。
2. 点击浏览器右上角的「SKU 采集器」。
3. 点击「采集当前页」。
4. 采集完成后，可以复制 CSV，也可以下载 CSV/JSON。

## 采集内容

插件会把“商品信息”里的内容拆成：

- 商品名称
- SKU
- 商品编号
- 品牌
- 商品链接
- 图片链接

同时会尽量读取页面表格里的每一列，例如：

- 类目
- 所属类目
- 机会判断
- 月销量
- 月销售额
- 均价
- 月销售额环比
- 月浏览量
- 日均销量
- 日均销售额

如果页面使用懒加载，保持「自动向下采集当前列表」打开，插件会向下滚动当前列表并合并已加载的行。

如果页面有分页，需要翻到下一页后再点一次「采集当前页」。
